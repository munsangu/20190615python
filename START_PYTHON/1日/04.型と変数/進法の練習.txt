2진수는 넷기초 ipv4의 주소체계

● 2진수 => 10진수
1101 => 2^3*1 + 2^2 *1 + 2^1 *0  + 2^0 *1
       => 8 + 4 + 0 + 1
       => 13 
110101 => 2^5*1 + 2^4*1 + 2^3*0 + 2^2 *1+ 2^1*0 + 2^0 *1
          => 32 + 16 + 0 + 4 + 0 +1
          => 53
● 10진수 => 2진수
48 => 110000
68 => 1000100 

● 16진수 => 2진수
F E 5 (16진수) => 15 14 5 (10진수) => 1111 1110 0101 (2진수) 
12DE =>1 2 13 14 => 0001 0010 1101 1111
● 2진수 => 16진수
1011 0110 0101 (2진수) => 11 6 5 (10진수) => B 6 5 (16진수)
1100 0010 1001 => 12 2 9 => C 2 9
